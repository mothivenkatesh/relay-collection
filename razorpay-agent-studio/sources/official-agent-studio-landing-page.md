# Official Agent Studio Landing Page

**Source:** [Razorpay Agent Studio | AI Agents for Payments & Revenue Operations](https://razorpay.com/agent-studio/)

## Key Points

- Positions Agent Studio around "intelligence on demand" for payments and revenue operations.
- Core premise: businesses lose revenue not only because payments fail, but because teams do not have time to fix what happens after.
- Describes AI agents that detect problems and take action on behalf of businesses.
- Offers three entry paths:
  - Customize a prebuilt agent.
  - Onboard as an AI partner.
  - Build an agent from scratch.
- States Agent Studio is built on Anthropic's Claude Agent SDK.
- Includes a quote from Irina Ghose, Managing Director, Anthropic India, about agents addressing commerce challenges such as revenue recovery, dispute resolution, and cashflow prediction.

## Agents Listed

- Dispute Responder
- Subscription Recovery
- Abandoned Cart Conversion
- RTO Shield
- RTO Insights
- Settlement Insights
- Cashflow Forecaster

## Workflows Mentioned

- Following up on unpaid invoices.
- Paying vendors and partners on schedule.
- Recovering failed or abandoned checkouts.
- Managing subscription renewals and recurring billing.
- Tracking settlements.
- Flagging suspicious transactions before they cause losses.

## Notes

The landing page is useful as the broad product positioning page. It emphasizes autonomous operational execution and partner/custom agent creation more than deep governance detail.
