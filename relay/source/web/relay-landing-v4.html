Platform capabilities

Everything a payment agent needs.
Nothing you have to build.

Relay is a full agent platform, not a notification tool. Every capability below is available to every agent you activate.

⚡

Trigger on any payment event

COD placed. Payment failed. Settlement run. Cart dropped. Refund initiated. Every Cashfree event is a trigger. Your agent decides what happens next.

🔀

Multi-step sequences with branching

Message at 15 min. Follow-up at 2 hrs. Escalate ops at EOD. If the customer replies, the sequence stops. Build logic once. It runs every time.

📡

Cross-channel in one agent

The same agent can send WhatsApp, log to Sheets, ping Slack, and send Gmail. No separate tools. No syncing. One agent, every channel.

📋

Full run history and audit log

Every agent run is logged. Every message tracked. Every outcome recorded. See what ran, when it ran, and what happened.

🔒

PCI DSS Level 1 infrastructure

Relay runs on Cashfree's own infrastructure. Your payment data never moves to a third-party server. What you send to channels is only what you configure.

🏠

Lives inside your Cashfree account

No new login. No new tool. No new data pipeline. Open Cashfree, go to Relay, activate. Your existing account is your agent platform.
