# ElevenLabs Connector and Case Study

**Source:** [Razorpay scales outbound merchant engagement with ElevenAgents](https://elevenlabs.io/blog/razorpay)

**Published:** April 7, 2026

## Summary

ElevenLabs published a customer story about Razorpay using ElevenAgents for outbound voice engagement. It is not solely about Agent Studio, but it directly confirms ElevenAgents as a connector on Razorpay Agent Studio.

## Key Points

- Razorpay deployed ElevenAgents for outbound voice agents in Hinglish.
- Use cases included churn recovery, feature adoption, and incident management.
- ElevenLabs reports 35,000+ calls completed across 5+ phase-one use cases.
- Reported connection rate was close to 28%, comparable to human call-center benchmarks.
- Post-call analysis extracts structured fields such as merchant intent, drop-off reason, and product interest from call transcripts.
- ElevenAgents is being brought as a connector on Razorpay Agent Studio.
- Agent Studio businesses can use conversational AI for subscription renewals, cart recovery, and customer re-engagement.

## Relevance

This gives concrete evidence for the voice-agent/connector layer behind Agent Studio and supports Razorpay's claim that agents can execute customer or merchant engagement workflows, not merely analyze dashboards.
