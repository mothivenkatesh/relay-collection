**Published:** March 12, 2026

**Summary:** Business Standard reported that Razorpay launched Agent Studio at FTX 2026 as an AI platform built on Anthropic's Claude technology. It also noted that Razorpay launched an agentic experience layer for onboarding, integration, and operational management.

## Coverage Pattern

Across media coverage, the recurring claims are:

- Agent Studio launched at FTX 2026.
- It is built using Anthropic's Claude Agent SDK.
- It is positioned as a B2B agent marketplace and builder platform.
- It targets payment and revenue operations use cases.
- Initial agents focus on abandoned cart recovery, dispute response, subscription recovery, and cashflow forecasting.
- The launch was paired with an Agentic Experience Platform.
