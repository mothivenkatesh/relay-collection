Help

Frequently asked questions

Relay is Cashfree's payment-native agent platform. It connects your Cashfree payment events — success, failure, refund, dispute — to the tools you already use: Slack, WhatsApp, Google Sheets, Calendly and more. Build agents in 30 minutes without code or a developer.
No. Relay has a visual drag-and-drop builder inside your Cashfree Dashboard — no terminal, no code. You can also use the Claude Desktop + MCP mode: connect the Relay MCP server and describe your agent in plain English. Claude configures it automatically.
Payment success, payment failure, payment dropped, refund initiated, dispute created, and simple schedule (hourly/daily). More triggers are on the roadmap: Shopify orders, customer cohort segments, incoming webhooks, and Google Drive file uploads.
Yes — Relay is free during beta. You can create and test agents immediately. Going live requires a one-time activation request, which we process quickly. Long-term pricing will be usage-based for certain agent types.
Relay is payment-native. Cashfree payment events (success, failure, refund, dispute) are first-class triggers with typed payloads — no webhook configuration, no manual field mapping. You're not building around payments; they're built in. Relay also runs execution securely in Cashfree's cloud, so the AI cost stays with your chosen model, not with us.
After testing your agent in sandbox against real Cashfree events, submit an activation request from the dashboard. We typically activate within 24 hours. Once active, the agent runs automatically on every matching event.

Get started

Live in 3 steps

Claude Desktop config

Add this to your claude_desktop_config.json

{
  "mcpServers": {
    "cashfree-relay": {
      "command": "npx",
      "args": ["@cashfree/relay-mcp"],
      "env": {
        "CF_APP_ID": "YOUR_APP_ID",
        "CF_SECRET": "YOUR_SECRET_KEY"
      }
    }
  }
}
What you can do once connected
  • 1
    Ask questions about your payments. "Show me all failed payments above ₹10,000 in the last 7 days."
  • 2
    Build agents by describing them. "When a payment fails, send a WhatsApp to the customer and log it to Sheets."
  • 3
    Manage your Relay agents. List, pause, or check run history without opening the dashboard.
  • 4
    Handle disputes. Draft evidence responses and route for approval — all via Claude chat.
